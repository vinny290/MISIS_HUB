1. 
   - **Клонировать репозиторий:**
     ```bash
     git clone https://github.com/vinny290/MISIS_HUB.git
     ```
2. 
   - **Запуск FrontEnd:**
     ```bash
     cd client
     npm install
     npm run dev
     ```
3. 
   - **Запуск BackEnd:**
     ```bash
     cd server
     npm install
     node app.js
     ```
